The non essential library contains nodes intended to be used with [PopcornFX](https://www.popcornfx.com/).

This library contains a variety of nodes and templates to help you in your projects. 
You will find :
- Math template (vector manipulation, coords manipulation, etc)
- Debug helpers
- Signal processing
- Particle spawn templates
- Function exposed but not templated by Persistant Studios
- Other misc stuff

In the future I plan to add :
- Asset generator templates
- Test meshes / test images
- Python script to generate graphs / Popcorn scripts
- Preset effects
- Demo/tutorial/unit test effects

## Installation
>You shouldn't download the repo itself if you intent to use the library in your project

First download the [latest release](https://github.com/Gus-The-Forklift-Driver/NonEssentialLibrary/releases/latest/download/Non.essential.library.pkkg)

To install, go to you project. In the content browser go to File -> Import Package, select the .pkkg you downloaded and it should install.

The nodes should be accessible straight away in your effects.

## Usage and documentation
There are no documentation for this project right now. All the nodes should be either self explanatory OR have a description on how they work.
